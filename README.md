# Robopop_Eve
Num estudo particular, desenvolvi um jogo usando Unity e C#, no qual os personagens e inimigos foram criados por uma talentosa criança de apenas 10 anos. Eu assumi a responsabilidade de criar a dinâmica do jogo.
